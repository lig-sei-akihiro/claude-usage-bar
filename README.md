# claude-usage-bar

![CI](https://github.com/lig-sei-akihiro/claude-usage-bar/actions/workflows/ci.yml/badge.svg) ![Swift](https://img.shields.io/badge/Swift-6-F05138?logo=swift&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-14%2B-000000?logo=apple&logoColor=white)

Claude Code の usage（5h セッション / 週次）をメニューバーに常駐表示する macOS アプリ。外部依存ゼロ、macOS 14+。

## インストール

```bash
brew install lig-sei-akihiro/tap/claude-usage-bar
```
